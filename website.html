Holiday Diorama / The Witch's Hut

⟡ Assignment work for FIT2092. To this day this remains one of my favourite projects! I was tasked to create a 3D space and environment using code, specifically HTML, CSS and Javascript.
⟡ This assignment helped me better understand Javascript, creating functions, using random attributes and 3D rotation values.
⟡ This encapsulates my love for art as well as the incredible flexibility of what can be done with and within code.

The Museum of Alternate Facts

⟡ Assignment work for FIT2092. The prompt was to create an online catalogue of history or facts that warp the truth. I decided to make an online museum of fake animals, hybrids I created using Clip Studio Paint.
⟡ This project features the most Javascript I had ever used at this point, and allowed for tons of experimentation, as well as including scrolling locks and progress trackers through the navigation buttons.
